# AGENTS.md

## 技术栈规范
- Node.js 22 LTS
- TypeScript（强制强类型）
- MongoDB（Mongoose）
- Express
- Zod（用于 DTO 校验）
- Jest（用于测试）

## 代码底线
- 所有关键业务流必须有中文注释，尤其是动态跳转、合法性校验、聚合统计链路。
- 严禁硬编码环境参数，数据库连接、JWT 密钥、端口等必须通过环境变量注入，并提供安全默认值。
- 绝对遵循零猜测铁律：所有核心字段、状态机、题型约束、跳转规则必须先在数据契约中定义，再在代码中实现。
- 发布态问卷禁止结构性修改，避免历史答卷与题目定义失配。
- 禁止依赖数组下标表达业务身份，题目与选项必须使用稳定 ID。
