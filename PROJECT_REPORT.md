# PROJECT_REPORT.md

## 项目目标

第一阶段的目标是完成一个可运行的在线问卷系统，覆盖以下能力：

- 用户注册与登录
- 问卷草稿创建、修改与发布
- 支持动态跳转的填写端
- 后端防伪造提交校验
- 统计结果聚合
- 前后端联调与交付级文档整理

最终交付结果不仅包含后端服务，也包含一个基于 React 的填写端前端，能够完成真实问卷加载与提交。

## 系统设计

系统采用前后端分离架构：

- 后端：`Node.js + Express + TypeScript + MongoDB`
- 前端：`React + Vite + TypeScript + Framer Motion`

架构上的核心原则是“体验逻辑与真实性裁决分离”：

- 前端负责渲染、输入、翻页与本地体验级推演。
- 后端负责问卷定义校验、路径重演、防伪造校验和统计聚合。

这样做的好处是：

- 前端可以做复杂交互，而不会影响数据真实性。
- 后端可以拒绝任何由前端篡改或伪造的非法答案。

## MongoDB 设计

数据库包含三个核心集合：

- `Users`
- `Surveys`
- `Responses`

其中最关键的是 `Surveys` 与 `Responses` 的嵌套结构设计：

- `Surveys.questions` 中直接内嵌选项、校验规则与逻辑跳转规则。
- `Responses.answers` 中直接内嵌每题答案。

采用 MongoDB 文档模型的理由是：

- 问卷天然适合以完整文档形式读取。
- 题目类型和校验规则存在可变字段，文档模型更灵活。
- 聚合统计可以直接对嵌套答案数组做 `$unwind` 和 `$group`，实现成本低。

## 跳转设计

跳转设计分两层：

- 前端 `survey-engine`：
  用 `replayFromStart` 从首题开始本地重演路径，决定当前题目、可达路径和幽灵答案集合。
- 后端 `responseService`：
  在 `/submit` 阶段重新从入口题开始重演整条路径，作为唯一真实路径。

具体规则如下：

1. 题目按 `order` 排序。
2. 从首题开始读取答案。
3. 按 `logicRules` 顺序找首个命中规则。
4. 若无命中规则，走 `defaultNextQuestionId`。
5. 遇到 `END` 结束路径。

这样既支持动态跳题，也保证路径可以被确定性复现。

## 校验设计

校验分为两大阶段：

### 1. 问卷定义校验

在创建、更新和发布问卷时，后端检查：

- 题目 ID 是否重复
- 题目顺序是否重复
- 跳转目标是否存在
- 是否存在自循环或图循环
- 是否存在不可达题目
- 不同题型的校验区间是否合法

### 2. 答卷提交校验

在提交问卷时，后端检查：

- 问卷是否存在
- 是否已发布、是否已截止
- 匿名权限是否满足
- 必答题是否缺失
- 答案类型是否合法
- 选项、长度、数值范围是否合法
- 是否存在幽灵答案与非法跳题

其中最关键的是幽灵答案校验：

- 若某答案不在服务端重演出的 `visitedSet` 中，则直接拒绝提交。

## AI 使用过程

本项目通过 Gemini 与 Codex 进行协同开发，AI 的主要价值体现在：

- 快速生成后端骨架和前端工程初稿
- 辅助设计前后端状态机与接口 DTO
- 输出 MongoDB Schema、聚合查询和测试骨架
- 辅助完成复杂 CSS 和页面动画

但 AI 也出现过偏差：

- 初期视觉稿存在廉价拟物化倾向
- 某些交互方案不符合最终审美目标

因此最终成品是在 AI 初稿基础上，由主理人持续纠偏、收敛和拍板后形成的。

## 遇到的问题及解决思路

### 1. 前端动画状态机与后端真实逻辑耦合风险

问题：

- 如果让动画直接决定路径，前端一旦状态错乱，提交数据就可能与真实跳题规则不一致。

解决思路：

- 将前端拆为“业务引擎层”和“演出层”。
- 前端体验只消费 `replayFromStart` 的结果，不定义真实性。
- 后端重新重演路径，作为最终裁决者。

### 2. 真实联调中的跨域与接口契约问题

问题：

- 浏览器端直接访问 `localhost:3000` 会面临开发期跨域问题。

解决思路：

- 前端通过 Vite 代理 `/api` 到后端，避免修改后端业务逻辑。
- 同时统一前端 API 客户端，直接解包后端标准响应格式。

### 3. 视觉方向多轮返工

问题：

- 初版前端曾出现“套皮网页”或“廉价拟物化”的问题，与高完成度目标不符。

解决思路：

- 保持状态机与业务逻辑不动，只重构表现层。
- 最终采用克制拟物化的“活页羊皮卷”方向，统一为骨白纸张、深海蓝墨水、浮雕选项与克制翻页动画。
